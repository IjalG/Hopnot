# Technical Specification

> Corresponds to spec v1.7 §1 and §4.1.

## Core Data Model

All memories are stored in a directed weighted graph $\mathcal{G} = (\mathcal{V}, \mathcal{E})$.

| Symbol | Meaning | Range |
|:---|:---|:---|
| $\mathcal{V}$ | Node set (memory atoms) | finite set |
| $\mathcal{E}$ | Directed edge set (associations) | $\subseteq \mathcal{V} \times \mathcal{V}$ |
| $\mathbf{b}_{v_i}$ | Base semantic vector (**frozen**) | normalized, never modified |
| $\mathbf{p}_{v_i}$ | Personal semantic bias (drift layer) | initially zero, $\|\mathbf{p}\| \le 0.5$ |
| $\mathbf{e}_{v_i}$ | Runtime effective vector | computed at retrieval, temporary |
| $L2_{ij}$ | Long-term edge weight | $[0, 1]$, floor $0.001$ |
| $L3_i$ | Node base potential | $[0, 1]$ |
| $c_{ij}$ | Edge confidence | $[0, 1]$ |
| $\text{type}_{ij}$ | Edge type | `ASSOC` / `NOT` |

**Runtime effective vector:**

$$\mathbf{e}_{v_i} = \text{norm}(\mathbf{b}_{v_i} + 0.1 \cdot \mathbf{p}_{v_i})$$

## Dual-Vector Architecture

Each node maintains two vectors:

- **Base vector b** — generated by the embedding model, permanently frozen. Serves as the semantic anchor.
- **Bias vector p** — initially zero, fine-tuned via drift during consolidation. Enables personalized memory.

## Retrieval Phase

### Effective Edge Weight

$$w_{ij} = \text{TypeWeight} \times L2_{ij} \times (0.6 + 0.4 \times L3_j^{\text{eff}}) \times \text{RefinePenalty} + L1_{ij} \times 0.3 + \varepsilon_{ij}$$

- $\varepsilon_{ij} \sim \mathcal{N}(0, 0.05)$
- $\text{TypeWeight} = -0.2$ if `type=NOT`, else $1.0$
- $\text{RefinePenalty} = 0.3$ if a refinement edge exists, else $1.0$
- Clipped: $\max(-0.2, \min(1, w_{ij}))$

### Seed Selection

1. **Coarse recall**: $\mathcal{C} = \{v_i \mid \langle \mathbf{e}_q, \mathbf{e}_{v_i} \rangle \ge 0.1\} \cup \mathcal{C}_{\text{exact}}$
2. **Chitchat gate**: matches `^(你好|谢谢|再见|...` → return empty
3. **Cold start**: no candidates → create new node, $L3=0.2$, activation $=1.0$, skip RWR
4. **Neighbor voting**: $\text{ContextScore} = s_i + 0.5 \times \text{avg\_neighbor} + 0.2 \times \text{recency}$
5. **Energy normalization**: $\alpha_i = \text{score}_i / \sum \text{score}_j$

### Random Walk with Restart (RWR)

$$\mathbf{s}_{t+1} = (1 - \rho) \cdot \mathbf{s}_t \times \mathbf{W} + \rho \cdot \mathbf{s}_0, \quad \rho = 0.4$$

- Transition matrix $\mathbf{W}$ is row-normalized, NOT edges excluded
- Path coherence: $S(P_k) = \frac{1}{k}\sum_{i=1}^k \prod_{j=1}^i (1 - \langle \mathbf{e}_{v_{j-1}}, \mathbf{e}_{v_j} \rangle)$
- Expansion decision: $\Psi = 0.25A + 0.35S + 0.20N^* + 0.20D^*$, threshold $0.45$

### Output Truncation

- `MaxTotalTokens` = 4096
- `ReservedTokens` = 512
- `MemoryContextTokens` = 3584

## Consolidation Phase

### Edge Decision Tree

| Branch | Condition | Operation |
|:---|:---|:---|
| 1 | Edge doesn't exist | Create: $L2=0.45,\ c=0.5$ |
| 2a | $c \ge 0.3$ (consistent) | Strengthen: $L2\text{+=}0.05,\ c\text{+=}0.05$ |
| 2b | $c < 0.3$ (dying) | Recover: $L2\text{+=}0.08,\ c\text{+=}0.10$ |
| 3 | Relation conflict | Downgrade $c\text{-=}0.03$, create NOT edge |
| 4 | More specific relation | Keep old edge, create `refines="关联"` edge |

### Bias Drift

- **ASSOC (attract)**: $\mathbf{p}_A \leftarrow \mathbf{p}_A + \lambda \cdot (\mathbf{b}_B - \mathbf{b}_A)$
- **NOT (repel)**: $\mathbf{p}_A \leftarrow \mathbf{p}_A - \lambda_{\text{not}} \cdot (\mathbf{b}_B - \mathbf{b}_A)$
- Only triggered by `origin="explicit_fact"`

### Triangular Closure

Triggered locally when $A \to B$ is created, checks $B$'s top 50 out-neighbors:

$$L2_{AC} = 0.5 \cdot \frac{L2_{AB} + L2_{BC}}{2}$$

### L3 Heat Update

$$L3 = 0.7 \times L3 + 0.3 \times (\text{deg\_ratio} + \text{freq\_ratio}) \times 0.5$$

- NOT penalty: $L3 \times (1 - 0.005 \times \text{count\_NOT})$, floor $0.05$
